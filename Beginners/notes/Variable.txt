
	# Variable คืออะไร ?
	variable คือตำแหน่งหนวยความจำที่มีชื่อบนแรม ที่ถูกจัดสรรชั่วคราวระหว่างการทำงานของโปรแกรม
		- ทุก ๆ variable จะถูกเก็บไว้ใน Stack
		- ตัวแปรที่ถูกประกาศไว้ใน method จะเรียกว่า local variable
	
	# ทุก ๆ ครั้งที่มีการเรียกใช้ method ก็จะถูกสร้างลงไปใน stack ใหม่

	# และเมื่อการดำเนินการนั้นใน Stack เสร็จแล้ว มันจะถูกลบโดยอัตโนมัติ

	# C# ต้องประกาศตัวแปรทั้งหมดก่อนการใช้งาน

	# Syntax	-> DataType VariableName;
