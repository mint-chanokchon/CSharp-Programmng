
	### Systax of C# Program
		- ทุกโปรแกรม C# จะต้องมี class อย่างน้อย 1 คลาส
		- ทุกโปรแกรมจะเริ่มทำงานที่ Main()
		- หาก Main() return 0; จะเป้นการบอกระบบปัฏิบัติการว่าโปรแกรมทำงานเสร็จแล้ว
		- หาก Main() return 1; จะเป้นการบอกระบบปัฏิบัติการว่าโปรแกรมมีข้อผิดพลาดบางอย่าง

	# Solution		คือ กลุ่มของโปรเจค
