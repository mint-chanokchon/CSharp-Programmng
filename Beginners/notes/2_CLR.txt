
    ### CLR (Common Language Runtime)

    # Class Loader
    มีหน้าที่โหลดคลาสลงไปในหน่อยความจำก่อน Execute โดยจะโหลดเฉพาะตลาสที่จำเป็น

    # Memory Manager
    มีหน้าที่จัดสรรหน่วยความจำสำหรับออบเจกต์และตัวแปรทั้งหมด
        - ตัวแปรจะถูกเก็บลงใน      -> stack
        - ออปเจ็กจะถูกเก็บลงใน     -> heap

    # Garbage Collector
    มีหน้าลบออบเจ็กที่ไม่ถูกใช้งานออกจากหน่วยความจำ หลังจากโปรแกรมทำงานถึงจุดสิ้นสุด method

    # JIT (Just-In-Time) Compiler
    มีหน้าที่แปรง MSIL Code เป็นภาษาเครื่อง โดยมันจะแปรงแค่ Code ที่ถูกโหลดมาจาก Class Loader 

    # Exception Manager
    ตรวจจับ error ที่เกิดขึ้นทั้งหมดระหว่างการ Execute และจะดำเนินการใน Catch block หากมี

    # Thread Manager 
    เป็นตัวจัดการเทรด รัน และ หยุด การทำงานของเทรด
    โดยปกติแล้ว .NET จะมีเทรดหลักอยู่แล้วที่ชื่อว่า Main Thread
    โดยเราสามารถออกแบบให้โปรแกรมใช้งานแบบ MultiThread ได้
    โดย Thread ทั้งหมดนี้จะถูกจัดกสนโดย Thread Manager

    # Security Manager
    ทำหน้าที่ตรวจสอบทุกคำสั่งก่อนการ Execute
    โดยเมื่อโปรแกรมพยายามเข้าถึงทรัพยากรณ์ hardware
